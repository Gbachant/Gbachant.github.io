Github-Pages Portfolio

February 2018

This site was built from scratch using Bootstrap. It is hosted though GitHub's lovely (and free) Pages.

Needing a portfolio website and still being somewhat new to web development I decided to build my own website. I initially planned on building a GitHub-pages website in markdown with a jekyll theme. I eventually decided that I would much rather have the full experience of building a website so I decided to dive right in and build it from the ground up in pure HTML, CSS and Bootstrap.
